Cypress-do-zero-a-nuvem


Documentação de projeto do Curso Cypress do zero a nuvem.


Pré requisitos:
É necessário possuir o Node.js e o NPM instalados na máquina.
Foi utilizado a versão v24.12.0 do Node e 11.6.2 do NPM. Recomendo utilizar a mesma versão ou superiores.


Instalação:
Execute o comando "npm install cypress".


Testes:    
Para inicializar os testes, é necessário abrir a tela iterativa do Cypress através do comando "npx cypress open"
